# 🌍 COSMOX - Travel Website

COSMOX is a simple multi-page **travel website** built using **HTML and CSS** as part of frontend practice.
This project demonstrates how to structure and organize a website with multiple pages like booking, destinations, safety, and support.

---

## 📌 Project Overview

The goal of this project is to practice creating a **multi-page website** with clean navigation and organized content.
Each page represents a different part of a travel platform.

---

## 📂 Pages Included

* `main.html` → Homepage / landing page
* `destinations.html` → Travel destinations
* `booking.html` → Booking section
* `safety.html` → Travel safety information
* `support.html` → Help & support page
* `training.html` → Training / guides section

---
